# ?? Agentic Healthcare Maps 
 
AI agent that turns 10,000 messy hospital records into a living intelligence network. 
 
## Problem 
 
In India, a postal code can determine a lifespan. Families travel hours only to find the help they need isn't there. 
 
## Solution 
 
- Ingest messy hospital records (PDFs, CSVs, handwritten notes) 
- Extract structured data: beds, specialists, ventilators, location 
- Triage symptoms  Recommend nearest hospital with capacity 
 
## Tech Stack 
 
- LLM: GPT-4 / Claude 
- Vector DB: Chroma 
- Map: Leaflet + OpenStreetMap 
- Backend: FastAPI 
- Frontend: React
